3-Legged OAuth

test.3leg - Reference Application

This application demonstrates the flow involved with a BlueJeans 3-Legged OAuth process.

  1. Clicking on the Request button below will send the browser to the /makerequest page.
    • This page causes the browser to redirect to the BlueJeans.com authorization page.
    • There you will be asked to login, and then approve this application (test.3leg) request to access your meetings
  2. The application sets your return from BlueJeans authorization to the /callback page.
    • If you declined access, this page will display a 401 error, along with an error message.
    • If you accepted then the callback will include an authorization code. This code is used in a BlueJeans OAuth API call to generate your access token.
